Digitalocean 400$ 学生包方式教程

2022-12-01 digitalocean ， 服务器

介绍📝

Digitalocean成立于2012年，总部在纽约，是一个比较知名的的云主机商家，采用KVM虚拟技术，配置高性能的SSD储存，服务器配备的是1000M端口，以512M内存为起点，月付最低至5美元，支持小时计费，随便换IP。

准备工具🛠

最重要的需要一个耐心
准备一个Github学生包账户
下载指纹浏览器：https://www.adspower.net,注册并登录
一张国内外任意余额有1.2美金的借记卡/信用卡（虚拟卡看脸）ps：切记不能一卡多号
一个国内网络上网环境(尽量与使用的卡注册地址同一地址)

点击注册后会弹出一个 github 授权页面，如下图 请点击绿色的按钮进行授权操作，如果没有弹出此页面，请重新从以上步骤操作一遍。

卡片验证页面为下图，请使用卡片注册地信息来填写，切记一定要使用卡片注册的ip来注册do，因为审核都是机器人，尽量要伪装真实用户环境注册。成功几率会很高.
